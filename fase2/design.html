Propósito

En la actualidad el uso de herramientas digitales es uno de los principales recursos, donde realizamos todo tipo de actividades cuyas ventajas, son que pueden ayudar a interactuar más con la tecnología de hoy en día, una de ellas es son las páginas Web. En esta sección haremos una síntesis de las etiquetas básicas de HTML, atributos y valores, para la construcción de cualquier sitio web.

Etiquetas Básicas de HTML

Empezar a adentrarse en el mundo de la programación web es una decisión que cada vez son más las personas que se dan cuenta de las oportunidades profesionales que supone contar con este tipo de conocimientos tan valorados por las empresas en la actualidad. Comprender el significado y la composición de las etiquetas HTML es el primer paso que debe dar cualquier profesional que aspire a convertirse en desarrollador o ingeniero web.

¿Qué son las etiquetas HTML? HTML es un lenguaje de marcado, lo que significa que está escrito en un código que es fácil de leer por cualquier persona, es decir, el texto de las páginas web está "etiquetado" con dichos códigos para indicar a los navegadores web cómo mostrar el texto.

Cuando codifica en HTML, está escribiendo marcado HTML. Todas las etiquetas HTML se componen de una serie de partes específicas, que incluyen: carácter "menor que" < Determina qué palabra de etiqueta o carácter introducir Cualquier número de atributos HTML que desee usar, en el formato nombre = "valor" carácter "mayor que" > El hipertexto (HTML) es el lenguaje que compone la mayoría de las páginas web y plataformas en línea. HTML no se considera un lenguaje de programación porque no puede crear una funcionalidad dinámica.

¿Cómo funcionan las etiquetas HTML? La mayoría de las páginas web tienen varias páginas HTML diferentes. Por ejemplo, página de inicio, página de producto, página de contacto, etc. Cada uno de estos tiene HTML separado. Los documentos HTML son archivos que terminan en .html o .htm. El navegador lee el archivo y muestra su contenido para que lo vean los usuarios de Internet. Los elementos HTML son los componentes básicos de las páginas web. Las etiquetas contienen mucha información. Le dice al navegador dónde comienza y termina cada elemento, y un atributo describe esas características. Los elementos HTML generalmente se dividen en tres partes. Según Equipo Vértice. (2009).
